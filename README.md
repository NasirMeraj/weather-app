# 🚀 JavaScript Practice Projects

Welcome to my JavaScript Practice Projects repository!  
This repository contains beginner-to-intermediate frontend projects built using **HTML**, **CSS**, and **JavaScript** to improve problem-solving, DOM manipulation, API handling, and overall web development skills.

---

## 📌 Projects Included

### 🌦️ Weather App
A simple weather application that fetches real-time weather data using a weather API.

#### ✨ Features
- Search weather by city
- Real-time temperature updates
- Weather condition display
- Responsive UI

#### 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- Weather API

---

### ❌⭕ Tic Tac Toe Game
A classic Tic Tac Toe game built with JavaScript and DOM manipulation.

#### ✨ Features
- Two-player gameplay
- Winner detection
- Draw detection
- Restart game functionality

#### 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
